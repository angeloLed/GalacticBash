**GalactixBash**
=======

A example Golang bash game inspirated at "Galactix" ( 1992 by Mountain King Studios ) usign [**termbox-go**](https://github.com/nsf/termbox-go) library

Usage
-----
run commands
```bash
go get
go run main.go
```
tested on ubuntu 16.04 and osx 10.11
