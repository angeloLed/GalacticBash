**GalactixBash**
=======

A sample Golang bash game inspirated at "Galactix" ( 1992 by Mountain King Studios ) usign [**termbox-go**](https://github.com/nsf/termbox-go) library

Usage
-----
run commands
```bash
go get
go run main.go
```
tested on ubuntu and osx
